- **Runtime:** Node.js 20+ / Bun

- **Real-time:** Socket.io

## 🚀 **Getting Started**

### **Quick Start (5 minutes)**
```bash
# Clone repository
git clone https://github.com/your-org/debugg.git
cd debugg/dashboard

# Install dependencies
npm install

# Setup environment
cp .env.example .env
# Edit .env with your settings

# Setup database
npx prisma migrate dev

# Start development
npm run dev
```
